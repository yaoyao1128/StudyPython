# 封装与解构

## 封装（装箱）

将多个值用逗号分割，组合在一起。

本质上返回的是一个**元组**，不过省略了小括号。

## 解构（拆箱）

把**线性结构**的元素拆开，并顺序赋给其他变量。

左边变量（无\*变量时）要与右边要解开的个数一样。

### Python3中新增的关于解构的操作

Python3中，可以使用`*变量名`表示多个变量，但在一次解构过程中只能使用一次，并且不能单独使用。

被`*`变量收集的变量最后会组成一个列表。

## 利用封装与解构交换元素

```python
a, b = b, a
```

等价于

```python
temp = a
a = b
b = temp
```
