# 5. 集合 \(set\)

## 什么是集合（set）

一种**非线性**的数据结构，是**可变的**、**无序的（散落）**、**不重复**的元素的**可迭代**的集合类型。

## set的定义（初始化）

```python
s1 = set()
s2 = {1, 2, 3}
s3 = set(range(5)) #{0, 1, 2, 3, 4}
s4 = set("abcabcde") #{a, b, c, d, e}
```

注意！`s5 = {}` 定义出来的是[**字典**](https://python3.ac.cn/2.-nei-zhi-shu-ju-jie-gou/6.-zi-dian)类型，不是set类型。
